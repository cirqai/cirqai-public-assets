# cirqai-public-assets

## Safe-Commit

- A chrome extension that helps you write safe and secure commit.
- Real-time SAST, secret, SCA, and SBOM scanning inside VS Code. Findings appear as squiggles and sidebar cards the moment you save.

### Installation
- Download [safe-commit](/safe-commit/safe-commit-1.0.1.vsix)
- Open command palette (`ctrl+shift+p` or `cmd+shift+p`)
- Run "Install from VSIX..."
- Select the downloaded file
- Reload VS Code
